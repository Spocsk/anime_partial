
# Symfony Moe API

API qui permet de créer un quiz avec des vidéos d'anime.





## Installation

Installer les librairies

```bash
  composer install
```

Lancer le serveur

```bash
  symfony serve
```

## Infos

Il n'y a pas de fixtures sur l'API. Il faudra donc insérer les questions directement avec l'application React.
